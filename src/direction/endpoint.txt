* Basic server starting api
http://localhost:5000/need-blood/api/v1

1. users endpoints
  * create user = /users/create-user
  require data = email, name, password, image
  * get all users = /users/all-users
  * get single data = /users/single-user/id
  * temporary delete =users/delete-temporary/id
  * permanent delete =users/delete-permanent/id

2. auth endpoints
  * login =/auth/login-user
  require data = email, password
